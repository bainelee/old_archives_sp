# 字体资源

## Sarasa-Mono-SC-Nerd

- **来源**：Sarasa-Mono-SC-Nerd（思源等宽 + Nerd Font 图标）
- **用途**：Figma 设计稿对应字体，详情面板等 UI 使用
- **文件**：
  - `Sarasa-Mono-SC-Nerd.ttf` - Regular（常规）
  - `Sarasa-Mono-SC-Bold-Nerd.ttf` - Bold（粗体）
  - `Sarasa-Mono-SC-Light-Nerd.ttf` - Light（细体）

## 项目引用

- `factor_details_panel.tscn`：通过 `detail_panel_theme.tres` 的 `default_font` 应用 Sarasa Regular
