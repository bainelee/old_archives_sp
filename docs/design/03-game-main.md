# 03 - 游戏主场景

## 概述

游戏主场景为项目的主运行入口（F5）。进入后展示 slot_0 地图，游戏运行期间无法唤出场景编辑器。

---

## 入口与隔离

- **进入方式**：打开 `game_main.tscn` 后使用「运行当前场景」(F6)
- **数据来源**：`user://maps/slot_0.json`
- **编辑器隔离**：主场景已留空，运行入口由当前打开的场景决定；进入游戏后无法唤出编辑器

---

## 功能

- 加载 slot_0 地图（底板 + 房间）
- 渲染底板（空/墙壁/房间底板）、网格线、房间底图、房间边框与名称
- 摄像机：中键拖拽平移、滚轮缩放

---

## 关键文件

| 文件 | 职责 |
|------|------|
| `scenes/game/game_main.tscn` | 场景根节点 |
| `scripts/game/game_main.gd` | 加载地图、渲染、相机控制 |
